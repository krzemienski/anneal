Anneal · Proposal C

Temper

Fixed-Point Deepen — one plan, heated and cooled repeatedly until scores stabilize.

Temper's defining property: the Red-Team Trinity runs at every depth. Cast runs red team once on a finished plan. Alloy runs red team once after synthesizer blending. Temper folds adversarial feedback into the very next rewrite. Each iteration genuinely learns from the prior.

The Pipeline

Convergence Rules

Three deterministic rules. No LLM decides when to exit. Exit when any one is true, in priority order variance > delta > cap.

RULE 1 · VARIANCE
variance(top-3 depth scores) < 0.3 — scores have stabilized. Order-insensitive, so a brief regression doesn't block convergence.
RULE 2 · DELTA
|score[-1] - score[-2]| < 0.15 — marginal improvement, diminishing returns. Absolute, so a regression also triggers.
RULE 3 · CAP
depth >= cap — hard iteration ceiling. Default 3, user-configurable via --depth N (range 1-5).

Roster at a Glance

Metis

Pre-plan consultant. Catches ambiguity and slop-risk before the planner runs.

Runs 1× per run

Prometheus-Temper

Writer. Seeds at depth 0; rewrites at N≥1 with prior Momus + Red Team folded in.

Runs 1× per depth

Deepen-Loop

Orchestrator. Tracks scores, invokes convergence-check, decides next depth.

Runs 1× per run · deterministic

Red-Team Trinity

Security · Scope · Assumptions. Parallel. Inline at EVERY depth.

Runs 3× per depth, parallel

Momus

Post-plan reviewer. Emits verdict AND 0-100 score. Score drives convergence.

Runs 1× per depth

Oracle

Bird's-eye synthesizer. Reads full depth history + convergence reason.

Runs 1× per run

Hephaestus

Functional validator. Builds and exercises the real artifact. No mocks.

Runs 1× per run · may re-loop

Atlas

Emitter. Writes XML + plan directory + depth-history.json on EMIT.

Runs 1× per run · only on EMIT

Invariants · The Non-Negotiables

INVARIANT T1 — Red team at every depth
The defining property. Remove inline red team and Temper degenerates into Cast-with-retry.
INVARIANT T2 — Momus emits 0-100 score
Required in Temper, not in Cast/Alloy. The score drives convergence.
INVARIANT T3 — Deterministic convergence
No LLM decides when to exit. convergence-check.py is a pure function over the score history.
INVARIANT T4 — Rewrites, not diffs
Each depth produces a full rewrite. Diffs let bad decisions persist invisibly.
INVARIANT T5 — Re-loop resets to Seed
On Hephaestus FAIL, the whole deepen loop restarts from depth 0 with failure folded into Metis directives.
