Anneal · Proposal A · Linear

Cast — single-pour plan review

One planner. One red team. One oracle. One validate. Emit. Cast is the fastest of the three Anneal architectures — the one you reach for when the spec is already clear and the task is scoped.

Version 0.1.0 Agents 7 named, 9 definitions Spawns ~8 per run Wall-clock ~4 min
The Pipeline

Seven stages, one direction

Stages 1 through 7 run sequentially. The only parallelism is at stage 5 where the Red-Team Trinity fans out. On validate FAIL, the run routes back through Metis — the failure folded as a new directive, not a fresh restart.

01
Intent Gate
Classify task shape. Reject unsafe inputs. Pass clean tasks forward.
02
Probe
Scan codebase. Enumerate skills in ~/.claude/skills/ and project skills. Read existing docs. Produce the probe report.
03
Enrich
Metis catches ambiguity and slop-risk patterns. Returns imperative directives the planner must follow. BLOCK with clarifying_questions aborts the run.
Metis
04
Plan — single pass
Prometheus writes the plan once. No tournament, no deepen loop. Momus audits. If Momus returns BLOCK, the re-loop folds the finding into a new Metis directive and restarts.
Prometheus-Cast Momus
05
Review — parallel adversaries
Red-Team Trinity fans out in parallel: Security, Scope, Assumptions. Three independent envelopes return; Oracle synthesizes the bird's-eye verdict. Oracle BLOCK forces re-loop.
Red-Team-Security Red-Team-Scope Red-Team-Assumptions Oracle
06
Validate
Hephaestus builds and exercises the real artifact. Captures evidence — build logs, CLI output, API responses, screenshots. Returns PASS or FAIL.
Hephaestus
FAIL → re-loop Failure folded back through Metis as a new constraint. Not a fresh restart — a constrained retry. Unbounded iteration count.
07
Emit
Atlas assembles the Opus 4.7 semantic-XML prompt and the plan directory, writing to ${ANNEAL_RUNS_ROOT:-./.anneal/runs}/{run_id}/. The only agent permitted to write outside the plugin's scoped directory.
Atlas
The Roster

Seven named agents

Each agent has a Greek etymology that signals when it runs. The trio at stage 5 is one role split into three independent spawns — parallelism is the independence guarantee.

Metis
Wisdom · Prudence
Pre-plan consultant. Catches ambiguity before the planner sees the task. Emits directives the planner must respect.
Stage 3 · Opus · 1x/iter
Prometheus
Forethought · Craft-fire
The planner. Single pass per iteration in Cast — no retry, no tournament, no deepen. Writes phase files and plan.md.
Stage 4 · Opus · 1x/iter
Momus
Satire · Mockery
Post-plan reviewer. Ruthless audit. Every phase gets at least one finding unless genuinely SAFE. Flags, never fixes.
Stage 4 · Opus · 1x/iter
Red-Team Trinity
Security · Scope · Assumptions
Three parallel adversaries. Independent envelopes, no cross-priming. BLOCK on CRITICAL findings forces re-loop.
Stage 5 · Sonnet · 3x parallel
Oracle
Delphic · Seer
Bird's-eye synthesizer. Reads every reviewer envelope. Emits release coherence, deployment risk, blast radius. Final gate before validate.
Stage 5 · Opus · 1x/iter
Hephaestus
Craftsman · Forge
Functional validator. Builds and exercises the real artifact. No mocks, no stubs, no test files. PASS or FAIL with cited evidence.
Stage 6 · Sonnet · 1x/iter
Atlas
Bearer · World-carrier
Emitter. Assembles XML prompt plus plan directory. The only agent permitted to write outside the plugin's scoped directory.
Stage 7 · Sonnet · 1x/emit
